# Quantum-Resistant Blockchain Security Toolkit

A comprehensive web application for Indian companies to migrate their blockchain and smart contract code from classical cryptography to post-quantum algorithms.

## Features

- 🔍 **Automated Cryptography Scanning**: Detect vulnerable classical algorithms (ECDSA, RSA, etc.)
- 🛡️ **Post-Quantum Migration**: Recommendations for ML-DSA/Dilithium and other NIST-approved algorithms
- ⛓️ **Blockchain Proof Storage**: Tamper-proof migration records on blockchain
- 📄 **Compliance Reports**: Professional PDF reports for RBI, CERT-In, and insurers
- 🏅 **Quantum-Safe Badges**: Verifiable certificates for regulatory compliance
- 🏭 **Sector-Specific**: Tailored for Rajasthan's manufacturing, pharma, healthcare, and banking sectors

## Technology Stack

- **Backend**: Python (FastAPI)
- **Frontend**: HTML5, TailwindCSS, JavaScript
- **Blockchain**: Web3.py (Ethereum/Polygon)
- **Cryptography**: PyCryptodome, liboqs-python
- **PDF Generation**: ReportLab
- **Code Analysis**: AST parsing, regex patterns

## Installation

```bash
cd quantum-blockchain-toolkit
pip install -r requirements.txt
```

## Usage

```bash
# Start the backend server
python backend/app.py

# Open frontend in browser
open frontend/index.html
```
